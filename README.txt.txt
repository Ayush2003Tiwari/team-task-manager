Team Task Manager (Full Stack MERN App)

Features:
- User Authentication (Login/Register)
- Create Projects
- Create Tasks
- Dashboard
- MongoDB Database
- REST APIs
- Role-based structure
- Fully deployed

Frontend:
React.js + Bootstrap

Backend:
Node.js + Express.js

Database:
MongoDB Atlas

Deployment:
Frontend: Vercel
Backend: Railway

https://team-task-manager-green-eight.vercel.app
